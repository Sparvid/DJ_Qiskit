# Qiskit examples for Deutsch-Jozsa (Presentation 2025-05-13)

## Installation
**Make a virtual environment**
```
python3 -m venv .venv
```
**Activate environment**
```
source ./venv/bin/activate
```

**Install requirements**
```
pip install -r requirements.txt
```

**Start notebook**
```
jupyter notebook
```
